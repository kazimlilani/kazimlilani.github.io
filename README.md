# Profile Website

A simple and modern profile website for GitHub Pages.

## Setup Instructions

1. **Add your profile picture:**
   - Place your profile picture in the root directory
   - Name it `profile.jpg` (or update the filename in `index.html`)

2. **Update your information:**
   - Edit `index.html` and replace:
     - "Your Name" with your actual name
     - "Your Title / Profession" with your title
     - Update all social media links with your actual profile URLs

3. **Deploy to GitHub Pages:**
   - Push this repository to GitHub
   - Go to Settings > Pages
   - Select your branch (usually `main` or `master`)
   - Your site will be live at `https://yourusername.github.io/repository-name`

## Customization

You can customize colors, fonts, and styling by editing `style.css`.
